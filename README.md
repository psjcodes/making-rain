# NEXRAD Radar Visualization

## Usage
Install requirements
```
pip install -r requirements.txt
```

Run Streamlit app
```
streamlit run app.py
```

Select a radar site
![radar site](img/radar_selector.png)

Wait for data to load
![load](img/load.png)

Scroll down to view data from past 3 hrs
![view](img/view.png)
